Creative Studio, a boutique design agency, wanted a portfolio website that would showcase their work in the best possible light while reflecting their minimalist design philosophy.

We created a clean, content-first website that lets the work speak for itself, with subtle animations that enhance rather than distract from the user experience.

### Key Features

- **Full-screen Gallery** — Immersive project showcases with high-resolution imagery
- **Smooth Animations** — Scroll-triggered transitions and parallax effects
- **Filter System** — Projects organized by category and medium
- **Contact Form** — Integrated inquiry system for potential clients

### Tech Stack

Built with Hugo for blazing-fast static site generation, with GSAP powering the animations and interactions.
